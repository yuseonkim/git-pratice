## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirements.txt
```

 
### How to Start

```shell
$python main.py
```

### Dependency

- Python=3.10
- django=3.0.0
- request==0.10.0

### Features

- 행운의 숫자 추출
- 행운의 숫자 저장
- 행운의 숫자 기반 구매
